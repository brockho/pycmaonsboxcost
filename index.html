Supplementary Material Webpage for the SBOX-COST 2023 Workshop Paper 249

This is the supplementary material webpage for the SBOX-COST 2023 paper

"Comparing Boundary Handling Techniques of CMA-ES on the bbob and sbox-cost Test Suites" by Dimo Brockhoff

as published with ACM in the GECCO 2023 Companion Proceedings.


Raw Data

  • CMA-ES with default BoundTransform on the bbob test suite: CMA-bt-bbob.tgz
  • CMA-ES with BoundPenalty on the bbob test suite: CMA-bp-bbob.tgz
  • CMA-ES with default BoundTransform on the sbox-cost test suite: CMA-bt-sboxcost.tgz
  • CMA-ES with BoundPenalty on the bbob sbox-cost suite: CMA-bp-sboxcost.tgz
All algorithms are also available directly via the postprocessing module of the COCO platform.

Postprocessed Data

  • Postprocessed data of all four data sets (as in the paper)
  • Postprocessed data of both boundary handling techniques on the bbob test suite
  • Postprocessed data of both boundary handling techniques on the sbox-cost test suite
